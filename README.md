# brother_bean_POS

## File Map

- [views/pages/login.html](views/pages/login.html) is the login view.
- [views/pages/admin.html](views/pages/admin.html) is the admin view.
- [views/pages/pos.html](views/pages/pos.html) is the cashier/POS view.
- [index.html](index.html), [login.html](login.html), [admin.html](admin.html), and [pos.html](pos.html) are compatibility redirects.

## Runtime Config

- [config/app.config.js](config/app.config.js) is the tracked runtime Firebase config used by the app at startup.
- [config/app.config.example.js](config/app.config.example.js) remains the reference template if the config ever needs to be regenerated.
